ADesign DNA

magasinn 在做什么把网站当成一本杂志：大留白、超大日西混排标题、内容块之间用横滚装饰带过渡，整体气质像 "印刷品 + 极少动态点缀"。
注意：本次迁移只借它的结构和动效，颜色仍沿用 portfolio 的暖米白 + 黄。

背景色（不换）
继续用 portfolio 的暖米白 #FAFAF8。比 magasinn 的灰更温暖、更贴"公益"题材气质。
字体系（借）
日西混排无衬线（YakuHanJP + dnp-shuei-gothic）。三个层级：
· 节标签 18px regular，纯英文，作为 eyebrow
· 主标题 48–80px 700，字距 -0.02em
· 正文 14–16px regular，行高 1.8
编号语言（借）
业务列表用半角括号 (01) (02) (03)；流程步骤同样 (01)–(06)。括号本身也是排版元素，绝不省略。
分栏（借）
不对称 grid。左列是节标签 + 节标题（占 1/3），右列是正文 / 卡组（占 2/3）。整页像翻一本左标右文的杂志。
卡片（借）
没有"卡片"。项目就是 一张图 + 标题 + 一句话，直接坐在底色上，靠间距而非边框来分隔。圆角极小或没有。
强调色（不换）
继续用 portfolio 的 #FFD60A 黄。用法上更克制——只点缀在 hero 高亮词、verdict 引用条左侧线，不大面积上墙。
BInteraction

交互方式magasinn 的交互全部围绕"滚动 = 翻页"展开，几乎没有点击展开/折叠，所有内容都铺开来读。

① Hero 自动横向轮播 + 索引指示器。顶部不是一张静态大图，而是 7 张图横向缓慢平移。右上角的 01 ─ 03 ─ scroll 一直在那儿告诉用户"现在第几张 / 总共几张 / 请向下滚"。

0103SCROLL ↓

② 无限横滚 marquee 文字带。section 之间用大字 PROJECT EDIT PROJECT EDIT… 或 NEXT NEXT NEXT… 做过渡。文字一直在缓慢向左移动，是页面唯一的"环境动效"。

NEXT ✕ NEXT ✕ NEXT ✕ NEXT ✕ NEXT ✕ NEXT ✕ NEXT ✕ NEXT ✕

③ Scroll-triggered reveal。所有内容块进入视口时统一 fade + up（24px），不分方向、不分类型，但每个子元素 stagger 0.1s 进入，让一整块有"逐字读"的节奏感。

④ 没有手风琴。这点和你当前 portfolio_v2 截然相反——magasinn 把所有项目细节都铺开，靠 大量留白和巨大标题 来切分章节，而不是靠 "+/×" 折叠。

CMotion

动效配方仅 2 个核心动画 + 1 个统一缓动函数，整站克制但高级。

缓动
cubic-bezier(0.16, 1, 0.3, 1)（ease-out-expo 的弱化版）。所有 fade/translate 都用这条。开头快、尾巴长 ≈ "丝滑减速"。
时长
1.6s 是统一的入场时长（比你现在的 0.6s 慢一倍多）。同一组元素用 0.1s / 0.2s / 0.3s ... 阶梯延迟。
Marquee
20–25s linear infinite，translateX(0 → -50%)，文字串复制一份接在后面做无缝循环。
Hover
图片 scale(1.02) + 0.6s ease-out-expo；链接下划线从 0 → 100% 用同款 0.6s 缓动。不用阴影变化。
Hero 入场
页面加载先有一个 0.7s 0.5s anime-loader-fadein 全局淡入，营造"翻杂志封面"的仪式感。
禁止项
不用 bouncy spring；不用 3D transform；不用 hover 抬升 + 投影；不用 fancy parallax。

FPriority

改造优先级按 ROI 排序，从最小改动就能换最大气质开始。

P0 · 1 小时
Patch 01（追加 tokens，不改原色）+ Patch 06（缓动统一 1.6s 加 ease-out-expo）。改完整站节奏立刻像换了一个人——但颜色完全没变，依旧是 portfolio 熟悉的米白 + 黄。风险最低。
P1 · 半天
Patch 03（CAPABILITY 去卡片，去白底圆角阴影）+ Patch 04（marquee 过渡带，深底用 #111，✕ 分隔符用 #FFD60A 黄）。这两条让页面"有节奏"，告别"卡片堆"。
P2 · 1 天
Patch 02（hero 编号 + scroll 指示）+ Patch 05（PROJECTS 长卷）。涉及结构调整，需重排已有手风琴里的 4–5 个子组件，需要时间。
慎做 · 不动
① UI 截图不换：你的项目图是流程截图/原型，本身就是内容的一部分，强行换成实拍会失真。② 色板不换：暖米白 + 黄是 portfolio 已经形成的视觉记忆，magasinn 的灰红只适合它的题材。

一句话总结：借 magasinn 的"杂志骨架"，留住 portfolio 的"暖色温度"。结构上换皮（去卡片、加 marquee、长卷化、统一缓动），但颜色、内容、信息密度全部不动。改完之后是一份"既有腔调、又有论据、还保留你原本辨识度"的设计师作品集。
